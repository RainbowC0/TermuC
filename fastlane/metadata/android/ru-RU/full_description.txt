TermuC — простая IDE для C/C++, работающая на базе мощного Termux.

<b>Установка</b>

Чтобы использовать все возможности IDE, выполните инструкции по настройке:

1. сначала установите Termux(https://f-droid.org/packages/com.termux);
2. выполните `pkg install clang' для установки компилятора clang & сервера языка clangd;
3. см. https://github.com/termux/termux-app/wiki/RUN_COMMAND-Intent#setup-instructions для разрешения вызовов из сторонних приложений;
4. затем установите TermuC.
